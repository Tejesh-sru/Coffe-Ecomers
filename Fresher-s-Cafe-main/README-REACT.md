# Fresher's Cafe - Professional React E-Commerce Application

A modern, professional coffee shop e-commerce website built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Modern React Architecture**: Built with React 19 and modern hooks
- **Responsive Design**: Fully responsive design that works on all devices
- **State Management**: Context API for global cart state management
- **Routing**: React Router for seamless navigation
- **Professional UI**: Beautiful, modern interface with Tailwind CSS
- **Shopping Cart**: Full-featured shopping cart with add, remove, and quantity update
- **User Authentication**: Login and signup pages with form validation
- **Product Catalog**: Grid-based product display with images and descriptions
- **Local Storage**: Cart persists across sessions

## 🛠️ Tech Stack

- **React 19** - UI library
- **Vite** - Fast build tool and dev server
- **React Router DOM** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Context API** - State management

## 📦 Installation

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Preview production build**:
   ```bash
   npm run preview
   ```
